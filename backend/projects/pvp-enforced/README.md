# PvP Enforced

**Division:** STUDIO
**Game Type:** OTSERV
**Engine:** TFS 1.5
**Language:** Lua/C++
**Protocol:** 8.60
**Status:** Em Desenvolvimento

## Description
Servidor OTServ no modo PvP Enforced — sem proteção de skulls, full loot,
pvp livre em qualquer zona. Focado na experiência de mundo aberto e disputas
territoriais. Público-alvo: jogadores veteranos que buscam alto risco e recompensa.

## Configurações Planejadas
- **PvP Type:** Enforced (sem proteção, sem skull branco)
- **Full Loot:** habilitado (drop de equipamentos ao morrer)
- **Protocol:** 8.60 (TFS 1.5)
- **Frags:** sistema de frags e banishment rigoroso para equilibrar o caos

## Features de PvP
- Sistema de war declarada entre guilds
- Controle de território por guild (houses em zonas disputadas)
- Kill ranking e leaderboard de PvP
- Sistema de bounty (recompensa por cabeça de player)
