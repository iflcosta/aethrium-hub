# Tibia Global 15+

**Division:** STUDIO
**Game Type:** OTSERV
**Engine:** Canary
**Language:** Lua/C++
**Protocol:** 12+ (cliente 15+)
**Status:** Em Desenvolvimento

## Description
Servidor estilo Tibia Global com suporte ao protocolo moderno 12+/15+.
Usa o engine Canary para compatibilidade com o cliente OTClient atualizado,
oferecendo uma experiência próxima ao Tibia oficial com melhorias e customizações.

## Diferenciais
- Suporte ao protocolo 12+ (exiva, exura vita, novas mechanics)
- Cliente OTClient com interface atualizada
- Conteúdo fiel ao Tibia Global (vocações, mapas, quests)
- Sistema de loot, bestiary e prey compatíveis com versões modernas
