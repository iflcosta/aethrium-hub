# Aethrium MMORPG

**Division:** STUDIO
**Game Type:** CANARY_MMORPG
**Engine:** Canary
**Language:** Lua/C++
**Status:** Em Desenvolvimento

## Description
MMORPG customizado com lore e sprites próprios, baseado no protocolo Tibia.
Desenvolvido sobre o engine Canary (fork moderno do TFS) com cliente OTClient customizado.

## Objetivo
Criar um mundo completamente original — continente, facções, criaturas, missões e economia —
diferenciado dos servidores OTServ tradicionais pelo lore exclusivo e identidade visual própria.
